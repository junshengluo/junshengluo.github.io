###8,域名解析,讲域名指向github Pages

  1. 在source文件夹中新建一个CNAME文件(无后缀名),然后用文本编辑器打开,在首行添加你的网站域名,如http://xxxx.com,注意前面没有http://,也没有www.

  2. 在域名解析提供商

    • 先添加一个CNAME,主机记录写@,后面记录值写上你的http://xxxx.github.io
    • 再添加一个CNAME,主机记录写www,后面记录值也是http://xxxx.github.io

这样别人用www和不用www都能访问你的网站.
也可以使用A记录,对应的记录值位github page的ip
